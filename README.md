# Sentiment_Analysis_IMDB_Dataset_
THIS HAS CODE AND DATASET FOR SENTIMENT ANALYSIS OF IMDB REVIEWS
THE DATASET COMPRISES 50K MOVIE REVIEWS (25K POSTIVE 25K NEGATIVE)
LOGISTIC REGRESSION MODEL USING SKLEARN KIT IS APPLIED
THE MODEL IS FIRST TRAINED AND THEN TESTED AND THE ACCURACY IS FOUND OUT TO BE 89.64%
