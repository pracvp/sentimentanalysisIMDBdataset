# sentimentanalysisIMDBdataset

LOGISTIC REGRESSION MODEL IS USED USING SKILEARN KIT
THE DATA IS FIRST TRAINED AND THEN TESTED
ACCURACY OF THE MODELIS FOUND OUT TO BE 89.64 %
